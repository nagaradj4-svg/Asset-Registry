Export Format

Sync — Computer ↔ Mobile

Move all data between devices using a single Excel file (records, photos, and dropdown lists included)

1. Export Sync Package FULL BACKUP

Creates one .xlsx file containing every asset (all fields, GPS, photos, and your Commune/Constituency/Village/Asset Type lists). Transfer it to the other device by WhatsApp, email, USB, or Google Drive, then use "Import" there.

2. Import Sync Package

Select a sync .xlsx file exported from another device. You'll see a preview before anything is changed on this device.

How it works

  • Each asset gets a permanent hidden ID (UID) the first time it's saved, so the same record is recognised on both devices during future syncs.
  • Importing merges by default: new records are added, and records that already exist (matched by UID) are updated with the imported version. Nothing else is touched.
  • Photos are preserved at full quality inside the sync file, so the export may take a few seconds for large registers.
  • Do this periodically (e.g. end of each field day) to keep the office computer and field mobile in agreement.
